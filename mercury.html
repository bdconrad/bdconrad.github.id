mercury.sh

Shell script written at GMMB(former employer) to completely automate the creation of a new WordPress site on Pantheon. Script creates a site, replaces default themes/plugins/content with GMMB standards, creates necessary users, clones locally, and opens up the Pantheon Dashboard in browser as well as the local codebase in Sublime Text.
